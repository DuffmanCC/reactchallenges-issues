# React Challenges — Bug Reports

This repository exists to collect bug reports and feedback for [reactchallenges.com](https://www.reactchallenges.com).

If you found a bug in a challenge, kata, quiz, or the platform itself, [open an issue](https://github.com/DuffmanCC/reactchallenges-issues/issues/new?labels=bug) and describe:

- What were you doing?
- What did you expect to happen?
- What happened instead?
- Browser and device info (if relevant)

The platform's source code is private — this is the public channel for reporting issues.
